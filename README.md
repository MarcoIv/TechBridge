# TechBridge
Final Project for COMP-3077

# Installation Guide
1. Download the Project folder.
2. Then you must create a SQL database with the following tables / relations:
   
<img width="662" height="461" alt="image" src="https://github.com/user-attachments/assets/95143ff0-d814-4d64-9db5-f304eee06e16" />

4. Then add your credentials to config.php to connect to that new database.
5. From there, add products to your database and they should automatically appear in the Catalogue of the site. To add product images, place the images in assets/products/ and ensure that they have the same name as the entry in the products table.
6. Add an admin user to your database as Admin level users can only be created in the database and not through the site to maintain security.
7. You can register a normal user with the register page and they will be added to the users table of the database.
8. You're all setup. Enjoy!
